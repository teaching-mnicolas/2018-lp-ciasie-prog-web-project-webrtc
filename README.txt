Démo application WebRTC:
	ouvrir deux navigateurs avec le fichier "index.html"
	Sur le 1er cliquer sur lancer le stream sur le second cliquer sur recevoir un stream
	Le premier navigateur va générer une "offre" qu'il faut copier et placer dans le champ de droite du second navigateur et cliquer sur enregistrer l'offre du second navigateur
	Le second navigateur va ainsi générer l'offre qui devra être copier dans la zone de texte droite du premier navigateur et cliquer sur enregistrer l'offre du premier navigateur
	ainsi on obtient une connexion entre les 2 navigateurs
